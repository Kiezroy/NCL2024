- Use ```hashcat -m 0 -a 3 ./hash.txt 'SKY-HQNT-?d?d?d?d'```
- ```-m 0``` is MD5 hash
- ```-a 3``` is brute force
- ```?d``` tells hashcat to use different digit in each place
